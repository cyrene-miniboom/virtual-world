# 血液微观漫游 GitHub 项目介绍页

这是基于《血液微观漫游》课程设计文档制作的 GitHub Pages 静态项目介绍网页。

## 文件结构

```text
blood-vessel-github-page/
├── index.html
├── styles.css
├── README.md
└── assets/
    ├── artery-capillary-vein.png
    ├── blood-cells-reference.png
    ├── coordinate-points.jpeg
    ├── island-master-plan.png
    └── demo-poster.svg
```

## 视频展示位

页面已经预留视频区域。把演示视频命名为：

```text
assets/demo.mp4
```

上传后无需修改网页代码，GitHub Pages 会自动在视频区域显示并播放。

## 部署到 GitHub Pages

1. 新建 GitHub 仓库。
2. 上传本文件夹中的 `index.html`、`styles.css` 和 `assets/`。
3. 进入仓库 `Settings → Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main / root`。
5. 等待 Pages 生成公开链接。

## 可替换内容

- 将导航或按钮链接替换成你的仓库地址。
- 将 `assets/demo-poster.svg` 替换成视频封面图。
- 将 `assets/demo.mp4` 替换成最终录屏或项目演示视频。
